üìñ How to Get a GitHub Personal Access Token

  1. Go to GitHub Settings ‚Üí Developer settings ‚Üí Personal access tokens ‚Üí Tokens (classic)
  2. Click "Generate new token (classic)"
  3. Give it a descriptive name like "LeetHub Extension"
  4. Select scopes:
    • public_repo (for public repositories)
    • repo (for private repositories)
  5. Click "Generate token" and copy it immediately
  6. Paste the token in the field above

Note: The repository must already exist on GitHub. LeetHub will not create it for you.
